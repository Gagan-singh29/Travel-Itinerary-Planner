HarryAI is an AI-powered travel itinerary planner that generates personalized
day-trip plans based on a city and user interests.

## Features
- Personalized day-trip itineraries
- Interest-based travel planning
- Fast responses using Groq LLM
- Clean Gradio interface
- Deployed on Hugging Face Spaces

## Tech Stack
- Python
- Gradio
- LangChain
- LangGraph
- Groq (LLaMA 3.3)
- Hugging Face Spaces

## How It Works
1. Enter a city
2. Enter your interests (comma-separated)
3. Get a curated itinerary for the day
